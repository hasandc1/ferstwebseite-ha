(HTML)

HTML steht für "HyperText Markup Language" und ist eine Auszeichnungssprache, die für die Strukturierung von Inhalten auf Webseiten verwendet wird. Es handelt sich um eine Grundlage des World Wide Web und bildet das Gerüst für den Aufbau von Webseiten. HTML verwendet eine Struktur aus sogenannten "Markup"-Tags, die den Browsern mitteilen, wie der Inhalt dargestellt und organisiert werden soll.

(CSS)

CSS steht für "Cascading Style Sheets" (Kaskadierende Stilblätter) und ist eine Stylesheet-Sprache, die verwendet wird, um das Aussehen und das Layout von HTML-Dokumenten zu gestalten. Mit CSS können Webentwickler das Erscheinungsbild von Webseiten kontrollieren, indem sie Stile für Elemente festlegen, wie z.B. Schriftarten, Farben, Abstände, Positionen und mehr.

(JAVA SCRIPT)

JavaScript (oft abgekürzt als JS) ist eine weit verbreitete Skriptsprache, die hauptsächlich für die Entwicklung von interaktiven und dynamischen Inhalten auf Webseiten verwendet wird. Im Gegensatz zu HTML (Hypertext Markup Language) und CSS (Cascading Style Sheets), die sich auf die Struktur und das Styling von Webseiten konzentrieren, ermöglicht JavaScript das Hinzufügen von Funktionalität und Interaktivität.
